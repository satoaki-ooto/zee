## ADDED Requirements

### Requirement: 矩形モードへの進入とアンカー固定

矩形選択モードに入ると、その時点のカーソル位置が矩形の **アンカー角**（行・visual column の両方）として固定されなければならない（MUST）。進入直後の矩形は「アンカー1点のみ」＝幅0・高さ1とする。進入は zi が Shift 修飾を持たないため、既存 `select-all` と同様の **多段プレフィックス方式** で行う。進入後にアンカーが移動してはならない（MUST NOT）。

#### Scenario: 進入でアンカーが固定される

- **WHEN** 通常編集状態で矩形選択モードに入る操作をする
- **THEN** その時点のカーソル位置（行・visual column）がアンカー角として固定され、矩形は幅0・高さ1になる

#### Scenario: 進入後もアンカーは動かない

- **WHEN** 矩形モードでカーソルを移動する
- **THEN** アンカーは進入時点の位置から動かず、対角だけが動く

### Requirement: 対角の決定と正規化

矩形モード中、通常の移動キーでカーソルを動かすと、カーソル位置が **対角の角** となり、アンカーとカーソルが作る (行範囲 × visual column 範囲) が現在の矩形になる（MUST）。行範囲はアンカー行とカーソル行の min..max（両端含む）、列範囲はアンカー列とカーソル列の min..max とし、左右どちら向きに選んでも正規化され向きに依存してはならない（MUST NOT depend on direction）。

#### Scenario: 対角移動で矩形が更新される

- **WHEN** 矩形モードでカーソルを上下左右に動かす
- **THEN** アンカーとカーソルが作る (行範囲 × visual column 範囲) が現在の矩形になる

#### Scenario: 選択の向きに依存しない正規化

- **WHEN** 同じ2点を、右→左／下→上の順で選ぶ
- **THEN** 確定する矩形領域は向きに依らず同一になる

### Requirement: EOL を越える列を許す

対角カーソルが、ある行では行末より右の visual column に来ることを許さなければならない（MUST allow）。矩形の右端列はカーソルの visual column で決まり、各行の実際の文字長には縛られない。

#### Scenario: 行末より右に対角が来る

- **WHEN** 対角カーソルを、ある行の行末より右の visual column に置く
- **THEN** 矩形の右端列はカーソルの visual column で決まり、短い行で右端に文字が無くてもエラーにならない（切り出し/削除の扱いは column-mapping の C-5 に従う）

### Requirement: キャンセルと非破壊

矩形モードでキャンセル操作（`C-g` 相当）をすると、矩形選択は破棄され、カーソル位置はそのまま通常編集状態に戻らなければならない（MUST）。矩形モード中はバッファ（Rope）を変更してはならない（MUST NOT）。矩形選択と通常の連続選択は両立せず、どちらか一方のみが active でなければならない（MUST）。通常選択（`begin_selection`）の振る舞いを書き換えてはならない（MUST NOT、C-3）。

#### Scenario: キャンセルで通常編集に戻る

- **WHEN** 矩形モードでキャンセル操作をする
- **THEN** 矩形選択は破棄され、カーソル位置は変わらず、バッファも変化しない

#### Scenario: 矩形モード中はバッファ不変

- **WHEN** 矩形モードでカーソルを動かして範囲を選ぶ
- **THEN** バッファ（Rope）は一切変更されない

#### Scenario: 通常選択と排他

- **WHEN** 矩形モードに入る
- **THEN** 通常の連続選択は active でなくなり、両者が同時に active になることはない

### Requirement: 幅0矩形を選択状態として許す

列範囲が空（行のみ）の幅0矩形は、選択状態として成立できなければならない（MUST）。padding は導入しない。幅0矩形に対するコピー/カットは no-op とする（詳細は copy / cut の各 spec）。

#### Scenario: 幅0矩形が選択状態になれる

- **WHEN** 列範囲が空のまま矩形モードで行方向にだけカーソルを動かす
- **THEN** 幅0・高さ>=1の矩形が選択状態として成立し、空白パディングは行われない
